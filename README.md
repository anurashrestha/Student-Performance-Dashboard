## Student Performance Dashboard

A clean and simple web application to input, track, and visualize student academic performance. Built with **Flask** and **Chart.js**, this dashboard helps users record grades, calculate averages, and display results visuallyâ€”making it a great portfolio project to showcase data handling, backend logic, and frontend design skills.

---

##  Features

-  Add student name, subject, and grade
-  Visualize performance using bar charts
-  Auto-calculate average and highest grade
-  JSON-based local data storage
-  Clean, responsive UI (Bootstrap)
